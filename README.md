# 🏛️ Mercado Central BH — Gestão de Contratos

Sistema de gestão de contratos comerciais desenvolvido para o Mercado Central BH.

## ✨ Funcionalidades

- Dashboard com KPIs e gráficos interativos
- Alertas automáticos de vencimento e reajuste
- Cadastro de contratos (Locação, Serviço, Publicidade)
- Importação de PDF e Excel
- Visualizador de documentos por contrato
- Checklist de renovação por contrato
- 100% client-side — sem backend, sem instalação

## 🚀 Como usar

Basta abrir o arquivo `index.html` no navegador.

## 🛠️ Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Chart.js — gráficos
- PDF.js — visualização de PDFs
- SheetJS — importação de Excel
- localStorage — persistência local
